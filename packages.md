# Packages & Repositories



## <a id="pkg_moss"></a>MOSS

Sparse multi-omic decomposition; on CRAN. Recognized among the top 40 R packages of 2020.

- **Language:** R
- **Source:** [https://github.com/agugonrey/MOSS](https://github.com/agugonrey/MOSS)
- **Install / docs:** [https://CRAN.R-project.org/package=MOSS](https://CRAN.R-project.org/package=MOSS)
- **Paper:** MOSS: multi-omic integration with sparse value decomposition (Bioinformatics 38 (10), 2956-2958, 2022)





## <a id="pkg_ecv"></a>eCV

Enhanced Coefficient of Variation and IDR extensions for reproducibility assessment of multi-replicate omic experiments. On CRAN. Methods paper on bioRxiv: https://www.biorxiv.org/content/10.1101/2023.12.18.572208v3

- **Language:** R
- **Source:** [https://github.com/eclipsebio/eCV](https://github.com/eclipsebio/eCV)
- **Install / docs:** [https://CRAN.R-project.org/package=eCV](https://CRAN.R-project.org/package=eCV)






## <a id="pkg_supp_moss"></a>GonzalezReymundez2021

Companion code for the MOSS paper (Bioinformatics 2022) — reproduces every figure and benchmark in the publication.

- **Language:** R
- **Source:** [https://github.com/agugonrey/GonzalezReymundez2021](https://github.com/agugonrey/GonzalezReymundez2021)

- **Paper:** MOSS: multi-omic integration with sparse value decomposition (Bioinformatics 38 (10), 2956-2958, 2022)
